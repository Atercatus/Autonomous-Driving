## Overview
* **Use raspberry pi 3**
* **Use 2 motors** 1 DC motor and 1 Servo motor
* **Behavior cloning** we use [The NVIDIA model](https://devblogs.nvidia.com/parallelforall/deep-learning-self-driving-cars/)
* **Object Detect** we use YOLO algorithm v3
